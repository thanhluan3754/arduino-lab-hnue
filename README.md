## Mục tiêu
Lưu trữ và chia sẻ các bài tập Arduino đã thực hành trên lớp và tự làm.

## Danh sách bài
- Bài 1: Điều khiển LED  
- Bài 2: Điều khiển LED 7 đoạn  
- Bài 3: ...  
- Bài 4: ...  

## Phần cứng sử dụng

### 1. Vi điều khiển và Board mạch
- Arduino Uno  
- ESP32 (sử dụng cho các bài WebServer và WiFi)

### 2. Module hiển thị
- LED đơn  
  - Mảng 5 LED  
  - Mảng 8 LED  
- LED 7 đoạn (loại đơn, anode/cathode)  
- Module 4 LED 7 đoạn  
- Module 4 LED 7 đoạn tích hợp IC 74HC595  
- Màn hình LCD 1602  
- Module giao tiếp I2C cho LCD  

### 3. Các loại cảm biến
- Cảm biến siêu âm HC-SR04  
- Cảm biến nhiệt độ và độ ẩm DHT11  
- Cảm biến chạm điện dung TTP223  
- Cảm biến chuyển động PIR  

### 4. Thiết bị nhập và điều khiển
- Nút nhấn (Push Button)  
- Bàn phím Keypad (4x4 hoặc 4x3)  
- Biến trở (Potentiometer)  
- Module Rơ-le (Relay)  
- Còi báo (Buzzer)  

### 5. IC bổ trợ và Giao thức
- IC chốt / dịch 74HC595  
- Giao tiếp Serial (UART)  
- Giao tiếp I2C  
- Giao tiếp SPI  
- Giao tiếp 1-Wire
